# 概要

Streamlitアプリの起動・確認・終了を管理するアプリです。

# 利用の前に

- 各種Streamlitアプリを同じフォルダ階層に準備してください。
  - [HDF Viewer](https://github.com/ishizawa2468/HDFViewer)
  - ..
- Pythonの環境構築を済ませてください。（ここから起動する先のアプリのライブラリも必要です）
